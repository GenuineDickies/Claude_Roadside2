# Master Roadmap — RoadRunner Admin

## Vision
RoadRunner Admin is a PHP-based roadside assistance & mobile mechanic operations platform providing end-to-end service management from intake through invoicing.

## Milestones

### Phase 1 — MVP Core (Current)
- [x] Authentication & session management
- [x] Customer management (CRUD)
- [x] Service request intake & tracking
- [x] Technician management & dispatch
- [x] Estimate creation workflow
- [x] Work order management
- [x] Invoice generation
- [x] Receipt tracking
- [x] Expense logging
- [x] Compliance/license tracking
- [x] Service catalog (React)
- [x] Production Director

### Phase 2 — Operational Excellence
- [ ] Reporting & analytics dashboard
- [ ] SMS/email notifications
- [ ] Mobile-optimized dispatch view
- [ ] Customer self-service portal
- [ ] Payment gateway integration

### Phase 3 — Scale
- [ ] Multi-location support
- [ ] Fleet management integration
- [ ] Inventory/parts tracking
- [ ] Route optimization

## Tech Stack
- **Backend:** PHP 8 + MySQL + PDO
- **Frontend:** Bootstrap 5.1.3 + Vanilla JS
- **Catalog:** React 18 (CDN)
- **Design:** Dark command center aesthetic
