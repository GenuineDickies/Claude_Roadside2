## MVP Requirements (v1.0)
### Must Have
- User authentication (login/logout)
- Customer CRUD with phone, email, address
- Service request creation with vehicle info and location
- Ticket numbering (RR-YYYYMMDD-NNNN)
- Technician profiles with skills and certifications
- Technician dispatch/assignment
- Estimate generation from service requests
- Work order creation from approved estimates
- Change order support
- Invoice generation with line items
- Receipt recording (payment tracking)
- Expense logging by category
- Compliance tracking (licenses, insurance, certs)
- Service catalog with categories and pricing
- Dashboard with KPIs

### Should Have
- Status workflow enforcement
- Audit trail for document changes
- Production Director for project management

## v2 Requirements
- SMS notifications

- Reporting with charts
- Payment gateway integration
