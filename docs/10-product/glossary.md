# Glossary

| Term | Definition |
|------|-----------|
| **Service Request (SR)** | Initial customer call/request for roadside or mechanic service |
| **Intake** | Process of capturing SR details: customer, vehicle, location, issue |
| **Dispatch** | Assigning a technician to an SR based on skills and availability |
| **Estimate** | Itemized cost projection for proposed work |
| **Work Order (WO)** | Authorized scope of work derived from an approved estimate |
| **Change Order** | Modification to an active work order (scope/cost change) |
| **Invoice** | Bill sent to customer for completed work |
| **Receipt** | Payment confirmation for a paid invoice |
| **Ticket Number** | Unique SR identifier: RR-YYYYMMDD-NNNN |
| **Document Chain** | SR → Estimate → WO → Invoice → Receipt inheritance flow |
| **Signal Colors** | Amber=pending, Green=success, Red=urgent, Blue=in-progress |
| **Navy Chrome** | #2B5EA7 used for buttons, links, nav — never for status |
